# 📊 Borsa Analiz Platformu

Bu proje, Borsa İstanbul (BIST) hisseleri için teknik analiz, veri takibi ve strateji geliştirme amacıyla geliştirilmiş bir analiz platformudur.

## 🚀 Özellikler

- 📥 Otomatik veri çekme (hisse, endeks vb.)
- 📈 Teknik analiz indikatörleri (EMA, MACD, RSI vb.)
- ⭐ Favori hisseleri takip edebilme (Watchlist)
- 📊 Günlük fiyat hareketleri analizi
- 🧠 Strateji test edebilme (Backtest)
- 🔔 Sinyal üretimi ve analiz

Bu tür platformlar genellikle yatırım kararlarını desteklemek için teknik göstergeler ve geçmiş verileri kullanır. Borsa İstanbul hisseleri üzerinde analiz yapmayı kolaylaştırır. (BIST Analiz)

---

## 🛠️ Kullanılan Teknolojiler

- Python / .NET / JavaScript (proje yapına göre düzenle)
- Veri çekme API’leri
- Grafik kütüphaneleri
- Teknik analiz algoritmaları

---

## 📦 Kurulum

bash git clone https://github.com/sanli-46/Borsa-Analiz.git cd Borsa-Analiz 

Gerekli paketleri yükle:

bash pip install -r requirements.txt 

---

## ▶️ Kullanım

Programı çalıştır:

bash python main.py 

veya

bash dotnet run 

---

## 📊 Örnek Analizler

- Hareketli ortalamalar (EMA)
- MACD kesişimleri
- RSI aşırı alım/satım bölgeleri
- Volatilite analizi

Bu tarz teknik analiz araçları yatırımcıların trendleri ve olası sinyalleri anlamasına yardımcı olur. (BIST Analiz)

- Kazanç / zarar oranı
- Risk analizi

yapabilirsiniz.
